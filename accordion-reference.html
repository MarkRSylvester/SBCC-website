Standard Accordion HTML Structure

Use this HTML structure for all accordion elements in your project:

<div class="accordion-item">
  <div class="accordion-header">
    <h3>Accordion Title</h3>
  </div>
  <div class="accordion-content">
    Content goes here
  </div>
</div>

Open-by-Default Accordion HTML Structure

For accordions that should be open when the page loads, use this structure:

<div class="accordion-item active open">
  <div class="accordion-header">
    <h3>Accordion Title</h3>
  </div>
  <div class="accordion-content" style="display:block; height:auto; max-height:none; padding:20px; overflow:auto;">
    Content goes here
  </div>
</div>

JavaScript Integration

Make sure to include our simplified accordion script at the end of your HTML:

<script src="/public/js/accordion.js"></script>

For debugging purposes, you can add this button to your page:

<button onclick="window.debugAccordions()">Debug Accordions</button>
