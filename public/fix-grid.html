Instructions:

  1. Drag the button above to your bookmarks bar
  2. Open the SBCC website
  3. Click the bookmark to activate the fix

What this does:

This bookmarklet will:

  • Find the "Beyond the Table" accordion
  • Replace its content with a 2x2 grid layout
  • Force the accordion to open

Alternative: Direct Console Fix

If the bookmarklet doesn't work, you can paste this code directly into your browser's JavaScript console:

var items = document.querySelectorAll('.accordion-item');
items.forEach(function(item) {
  var header = item.querySelector('h3');
  if (header && header.textContent.trim() === 'Beyond the Table') {
    var content = item.querySelector('.accordion-content');
    if (content) {
      content.innerHTML = '<div style="display:grid;grid-template-columns:1fr 1fr;gap:20px"><div style="background:#fff;padding:20px;border:1px solid #eee"><h4 style="color:#1D3557">Wine Pairing</h4><ul><li>Expert wine recommendations</li><li>Curated selections from vineyards</li></ul></div><div style="background:#fff;padding:20px;border:1px solid #eee"><h4 style="color:#1D3557">Table Setting</h4><ul><li>Elegant table design</li><li>Premium linens and dinnerware</li></ul></div><div style="background:#fff;padding:20px;border:1px solid #eee"><h4 style="color:#1D3557">Cooking Lessons</h4><ul><li>Hands-on instruction</li><li>Professional methods</li></ul></div><div style="background:#fff;padding:20px;border:1px solid #eee"><h4 style="color:#1D3557">Produce Boxes</h4><ul><li>Local organic farms</li><li>Artisanal specialty items</li></ul></div></div>';
      item.classList.add('active');
      item.classList.add('open');
      content.style.display = 'block';
    }
  }
});
